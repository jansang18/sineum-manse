# 웹 배포 안내 — 신의 음성 만세력 (PWA)

이 `web` 폴더 전체를 웹에 올리면, 브라우저에서 주소(URL)로 바로 접속할 수 있습니다.
설치형 PWA라 모바일에서 "홈 화면에 추가"하면 앱처럼 전체화면으로 실행되고, 오프라인에서도 동작합니다.

## 폴더 구성
- `index.html` — 앱 본체
- `polish.css` — 디자인 보강 스타일
- `manifest.webmanifest` — PWA 설정(이름/아이콘/색)
- `sw.js` — 서비스워커(오프라인 캐시)
- `icon-192.png` / `icon-512.png` / `apple-touch-icon.png` — 아이콘

> ⚠️ 서비스워커(오프라인·설치)는 **https 또는 localhost**에서만 동작합니다.
> 파일을 더블클릭(file://)으로 열면 화면은 보이지만 설치/오프라인 기능은 비활성화됩니다. 아래 호스팅을 쓰세요.

---

## 방법 A. Netlify Drop (가장 쉬움 · 1분 · 로그인 없이 시작)
1. 브라우저에서 https://app.netlify.com/drop 접속
2. 이 **`web` 폴더를 통째로 끌어다 놓기**
3. 즉시 `https://랜덤이름.netlify.app` 주소 발급 → 바로 공유 가능
4. (선택) 무료 가입하면 주소를 원하는 이름으로 바꾸고 영구 보관

## 방법 B. GitHub Pages (안정적인 주소 · 무료 · 깃허브 계정 필요)
계정(jansang18)이 있으시니, 원하시면 제가 저장소 생성·업로드·Pages 설정까지 도와드립니다.
완료되면 `https://jansang18.github.io/저장소이름/` 형태의 고정 주소가 생깁니다.

## 방법 C. Cloudflare Pages / Vercel
무료 가입 후 이 `web` 폴더를 업로드(또는 깃 연동)하면 됩니다.

---

## 업데이트 방법
앱 내용(`index.html` 등)을 수정한 뒤, 같은 폴더를 다시 올리면 됩니다.
서비스워커 캐시 버전을 올리려면 `sw.js`의 `sineum-manse-v1`을 `v2`로 바꾸세요.
